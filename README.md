# SSM_Frame

 **此项目供本人学习使用：**
        
   #### 1.安装依赖
       
       本地需要安装jdk、maven,redis，mysql数据库 等 基础工具
       
   #### 2.项目介绍
      
        本项目主要通过整合框架SSM(即：spring + spring mvc + mybatis)， 前端页面采用bootstrap框架,
        来写一个购物项目，mybatis分页插件Pagehelper使用。
        
        ** 采用maven分模块编程
        
        数据库名：MyLearn_3     表结构文件看：MyLearn_3.sql
